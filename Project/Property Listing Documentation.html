Using the Property Listing Application‌


About the application‌


The Property Listing application I a software tool for recording and searching data related to properties based in Dublin City, Republic of Ireland.

Learning to use this software is considerably easy and would not require the knowledge of any particular terminology.

Exploring the first (start) page‌

The first (start) page is the main page, what provides a hub for all the input and output features.



Input fields and their functions‌


  • House Number: to input new value, click into the text-box and enter the selected numerical value

  • Street: to input new value, click into the text-box and enter the selected value

  • Area: to select new value, click onto the down arrow on the combo-box and select the appropriate value from the dropdown list

  • District: to select new value, click onto the down arrow on the combo-box and select the appropriate value from the dropdown list

  • Type: to select new value, click onto the down arrow on the combo-box and select the appropriate value from the dropdown list

  • Heating: to select new value, click onto the down arrow on the combo-box and select the appropriate value from the dropdown list

  • Room: to select new value, click onto the down arrow on the combo-box and select the appropriate value from the dropdown list

  • BER Rate: to select new value, click onto the down arrow on the combo-box and select the appropriate value from the dropdown list

  • Size: to input new value, click into the text-box and enter the selected numerical value

  • Price: to input new value, click into the text-box and enter the selected numerical value

  • Facilities: to select from listed facilities, click on the appropriate check-box, to unselect re-click on the relevant check-box

    Record exploration‌


  • First: by clicking this button the application shall display the first record of the collection

  • Previous: by clicking this button the application shall display the previous record to the current one in the collection

  • Next: by clicking this button the application shall display the next record to the current one in the collection

  • Last: by clicking this button the application shall display the last record of the collection


    Record management‌

  • Add Record: by clicking this button the application will add the current record to the collection

  • Update Record: by clicking this button the application will update the current record in the collection with the modifications made

  • Delete Record: by clicking this button the application will delete the current record from the collection

  • Clear Records: by clicking this button the application will erase all records from the collection

  • Clear: by clicking this button the application will clear all fields on the current record

  • Cancel: by clicking this button the application will undo all changes made to the current record


    Opening, Closing and Saving file‌


  • Open File: by clicking this button the user is able to navigate to compatible (*.ser) saved file and open it

  • Close File: by clicking this button the user is able to navigate to a saved file and open it (no filter for compatible file formats)

  • Save As: by clicking this button the user is able to save the collection to a file (*.ser) while navigating in the file directory

  • Save: by clicking this button the user is able to save the collection to the created file (*.ser)

  • Save As CSV: by clicking this button the user is able to save the collection to a CSV file while navigating in the file directory


    Alternative functions‌


  • Search Record: by clicking this button the application will open the second page, where the user is able to search the collection by certain given criteria

  • Exit: by clicking this button the application will close without saving any the collection or any modifications

  • Guiding Mode: by default, the on-radio button is checked, what is activating the toolTip message for every functionality on the page; these will appear when the user hovers the mouse over the objects. To deactivate this, click the off-radio button

  • Index box: this area indicates the index of the current record

  • Back: by clicking this button, the application will return to the first page


General handling guide‌


Creating, saving (*.ser, *.csv) and updating own collection‌


To create your own collection, load the application > click Clear Records button > fill all fields with the appropriate values > click Add Record button. To create additional records, after clicking Add Record button > click Clear button > fill all fields with the appropriate values > click Add Record button.


To save (*.ser) collection for future use, after creating your own collection > click Save As button > navigate to the preferred directory > enter preferred file name in the File name text box > click on Save button.


To save (*.csv) collection for future use, after creating your own collection > click Save As CSV > navigate to the preferred directory > enter preferred file name in the File name text box > click on Save button.


To update your collection, load application > open your own file, or create your own collection > navigate to the relevant record > change the value of the appropriate field(s) > click on Update Record button.


Delete single record‌


To delete a single record, navigate to the relevant record > click on Delete Record button.


Search collection‌


To search collection, if the current location is the main page > click on Search Record button > fill the relevant field(s) with the appropriate value > click on Search button.

If the current location is the second page > fill the relevant field(s) with the appropriate value > click on Search button.
